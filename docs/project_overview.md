# Project Overview

## Objective

The goal of this project is to trqain a reinforcment learning (RL) agent to safely navigate a highway environment.

The agent will learn to:

-Maintain safe driving behavior
-Avoid collisions
-Change lanes when appropriate
-Maximize travel efficiency
-Operate in simulated traffic

## Technologies

-Python
-Gymnasium
-Highway-env
-Stable-Baselines3
-PyTorch
-Git
-GitHub

## Project Phases

### Phase 1
Create baseline random agent

### Phase 2
Train DQN reinforcement learning agent.

### Phase 3
Evaluate trained agent.

### Phase 4
Generate performance metrics and graphs

### Phase 5
Record demonstrations.

### Phase 6
Finalize documentation and portfolio presentation.

-Source code
