#RL Self-Driving Car Simulation

## Overview

This project explores reinforcement learning for autonomous driving using the Highway-env simulation environment

The objective is to train an agent capable of making safe driving decisions while navigating highway traffic.

## Technologies

-Python
-Gymnasium
-Highway-env
-Stable-Baselines3
-PyTorch
